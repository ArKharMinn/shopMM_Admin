To implement pagination in Vue.js with data fetched from a Laravel backend, you can follow these steps:

  1. Update Laravel Backend: Modify your Laravel controller to support pagination.
php
// app/Http/Controllers/PostController.php
namespace App\Http\Controllers;

use App\Models\Post;
use Illuminate\Http\Request;

class PostController extends Controller
{
    public function index()
    {
        $posts = Post::paginate(10); // Change 10 to your desired number of items per page
        return response()->json($posts);
    }
}
  1. Update Vue.js Frontend: Update your Vue component to handle pagination.
html
<!-- resources/js/components/PostList.vue -->
<template>
  <div>
    <div v-for="post in posts" :key="post.id">
      <h2>{{ post.title }}</h2>
      <p>{{ post.body }}</p>
    </div>
    <button @click="fetchNextPage" :disabled="currentPage >= totalPages">Next</button>
  </div>
</template>

<script>
export default {
  data() {
    return {
      posts: [],
      currentPage: 1,
      totalPages: null,
    };
  },
  mounted() {
    this.fetchPosts();
  },
  methods: {
    fetchPosts() {
      axios.get(`/api/posts?page=${this.currentPage}`).then(response => {
        this.posts = response.data.data;
        this.totalPages = response.data.last_page;
      }).catch(error => {
        console.error('Error fetching posts:', error);
      });
    },
    fetchNextPage() {
      if (this.currentPage < this.totalPages) {
        this.currentPage++;
        this.fetchPosts();
      }
    }
  }
}
</script>

In this example, we're using Laravel's pagination feature to fetch a paginated list of posts. The fetchNextPage method increments the currentPage and fetches the next page of posts from the Laravel backend.

Make sure your Laravel backend is set up to handle pagination properly, and adjust the frontend code as needed based on your pagination configuration.

To fetch quiz questions from your Laravel backend and display them one by one in a Vue.js component, you can modify your Laravel controller to return the quiz questions as paginated data. Then, in your Vue.js component, you can fetch the quiz questions and display them one at a time, advancing to the next question when the user clicks a "Next" button. Here's how you can achieve this:

Laravel Backend (Controller):

php
<?php

namespace App\Http\Controllers;

use App\Models\Exam;
use Illuminate\Http\Request;

class ExamController extends Controller
{
    public function quiz(Request $request)
    {
        $categoryId = $request->id;
        $quiz = Exam::where('category_id', $categoryId)->inRandomOrder()->paginate(1);
        
        return response()->json([
            'quiz' => $quiz
        ]);
    }
}

Vue.js Component:

html
<template>
  <div>
    <div v-if="currentQuestion">
      <h3>Question {{ currentIndex + 1 }} of {{ totalQuestions }}</h3>
      <h2>{{ currentQuestion.description }}</h2>
      <div v-for="(option, index) in currentQuestion.options" :key="index">
        <input type="radio" :id="'option_' + index" name="answer" :value="option">
        <label :for="'option_' + index">{{ option }}</label>
      </div>
      <button @click="nextQuestion">Next</button>
    </div>
    <div v-else>
      <p>No more questions</p>
    </div>
  </div>
</template>

<script>
import axios from 'axios';

export default {
  data() {
    return {
      quiz: [],
      currentIndex: 0
    };
  },
  computed: {
    currentQuestion() {
      return this.quiz[this.currentIndex];
    },
    totalQuestions() {
      return this.quiz.length;
    }
  },
  methods: {
    fetchQuiz() {
      axios.post("/api/quiz", { id: 1 }) // Pass the category ID as needed
        .then(response => {
          this.quiz = response.data.quiz.data;
        })
        .catch(error => {
          console.error('Error fetching quiz:', error);
        });
    },
    nextQuestion() {
      if (this.currentIndex < this.quiz.length - 1) {
        this.currentIndex++;
      }
    }
  },
  mounted() {
    this.fetchQuiz();
  }
}
</script>

In this setup:

  • The Laravel controller returns paginated quiz questions, fetching only one question per request.
  • In the Vue.js component, the fetchQuiz method fetches the quiz questions from the Laravel backend.
  • The currentIndex variable tracks the index of the current question being displayed.
  • The currentQuestion computed property retrieves the current question from the quiz array based on the current index.
  • The nextQuestion method advances to the next question when the user clicks the "Next" button.
  • The template displays the current question and provides a "Next" button to move to the next question.

With this setup, each click on the "Next" button will display the next question fetched from the Laravel backend.
